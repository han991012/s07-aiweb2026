# 📊 코스피 옵션 AI 브리핑

> 영업일 하나 입력만으로 — 그날의 코스피200 옵션 시장이 한눈에.

## ✨ 한 줄 요약

영업일 날짜 하나를 입력하면 KRX 한국거래소 OpenAPI에서 그날 코스피200 옵션(주식옵션 外) 일별매매정보를 가져와, GPT-4o-mini가 비전문가도 이해 가능한 한국어 시황 리포트로 변환해주는 데모. 전체 데이터 표 + 시장 요약 + PDF 다운로드까지 제공한다.

## ❓ 문제 (왜 만들었나)

코스피200 옵션 시장은 매일 수백 종목의 매매 데이터가 발표되지만, 표 형태의 숫자만 봐서는 시장 분위기를 파악하기 어렵다. 특히 옵션 도메인 지식이 약한 일반 투자자는 "거래량 5만, IV 22%" 같은 수치만으로 시장 흐름이나 헤지 수요 변화를 직관적으로 읽어내지 못한다. 매일 누군가가 사람 말로 풀어주면 좋겠지만 현실적으로 어렵다. AI가 이 역할을 대신해줄 수 있다.

## ⚡ 핵심 기능

### 1. 날짜 입력 → 전체 옵션 데이터 표로 표시
영업일을 입력하면 KRX OpenAPI에서 옵션 일별매매정보를 호출해 모든 종목의 데이터(종목명·종가·대비·IV·익일정산가·거래량·거래대금·미결제약정)를 상품/콜풋 필터가 가능한 표로 화면에 표시한다.

### 2. 시장 요약 통계 (콜/풋 세분화)
pandas로 총 거래량과 총 거래대금을 상단에 계산하고, 그 아래 권리유형별(콜/풋)로 거래량·거래대금·평균 IV·미결제약정을 세분화해서 보여주며, 거래량 상위 5개 종목을 콜·풋 각각 추출해 표시한다.

### 3. AI 시황 리포트 + PDF 다운로드
GPT-4o-mini가 위 통계치를 기반으로 비전문가도 이해 가능한 3~5문단 한국어 시황 리포트를 생성하고, 한글 폰트가 임베딩된 PDF로 다운로드받을 수 있게 한다.

## 🔄 동작 흐름

```
영업일 날짜(YYYYMMDD)
    ↓
KRX OpenAPI 호출 (옵션 일별매매정보)
    ↓
pandas로 통계 사전 집계 (콜/풋 분리 · TOP 5)
    ↓
GPT-4o-mini에 통계 요약 전달
    ↓
시장 요약 카드 + 콜/풋 세분화 + TOP 5 + 전체 표 + AI 리포트 + PDF 다운로드
```

## 🧰 기술 스택

- **UI**: Gradio (Python 단일 파일 앱)
- **LLM**: GPT-4o-mini (OpenAI API)
- **데이터 API**: KRX 한국거래소 OpenAPI
- **데이터 처리**: pandas
- **PDF 생성**: reportlab (한글 폰트 임베딩)
- **배포**: HF Spaces (Docker 모드)
